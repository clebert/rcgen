# @rcgen/generators

[![Package Version][package-badge]][package-npm]

A collection of higher-order generators for use with **rcgen**.

## Installation

### Using Yarn

```sh
yarn add @rcgen/generators
```

### Using npm

```sh
npm install @rcgen/generators
```
