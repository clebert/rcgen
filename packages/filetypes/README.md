# @rcgen/filetypes

[![Package Version][package-badge]][package-npm] [![API][api-badge]][api]

A collection of common file types for use with **rcgen**.

## Installation

### Using Yarn

```sh
yarn add @rcgen/filetypes
```

### Using npm

```sh
npm install @rcgen/filetypes
```
