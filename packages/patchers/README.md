# @rcgen/patchers

[![Package Version][package-badge]][package-npm] [![API][api-badge]][api]

A collection of convenient patcher functions for use with **rcgen**.

## Installation

### Using Yarn

```sh
yarn add @rcgen/patchers
```

### Using npm

```sh
npm install @rcgen/patchers
```
