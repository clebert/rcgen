# @rcgen/configs

[![Package Version][package-badge]][package-npm] [![API][api-badge]][api]

A collection of configuration providers for use with **rcgen**.

## Installation

### Using Yarn

```sh
yarn add @rcgen/configs
```

### Using npm

```sh
npm install @rcgen/configs
```
